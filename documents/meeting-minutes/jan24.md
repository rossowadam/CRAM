## Some thoughts:
Keeping Modularization gives more freedom to the user base but has extreme con where the user can do just about anything. A structured notes folder for one course could end up in a chaotic mess.
Enforcing structure by prepopulating known units and overarching topics while only permitting note writing reduces privilege to the user base as it should. Has the con where it limits a lot of freedom. 
A middle ground could be where we do prepopulate units/sections but also allow new subfolders through approval of a Prof/TA/Mod.
